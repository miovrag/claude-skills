# Bourdieu *Distinction* — Analytical Framework Skill

## Purpose

This skill produces a **structured analytical report** that applies Bourdieu's framework from *Distinction* to a described social phenomenon, group, or practice. The analysis is grounded in the original text via the reference file.

## Step 1 — Load the Conceptual Reference

Before writing any analysis, read the full reference file (relative to this skill's directory):

```
references/concepts.md
```

This file contains:
- All eight core concepts with direct textual grounding
- An analytical checklist (8 questions) to structure the report
- Page references and key quotes for citation

## Step 2 — Clarify Input (if needed)

The input is typically a description of a social phenomenon, group, or practice (e.g., "navike čitanja srednje klase u Srbiji", "crossfit kultura u Beogradu", "potrošnja organskih proizvoda").

If the input is ambiguous about:
- **Who** — which group or class fraction is the subject
- **Where** — national/local context matters for capital conversion rates
- **What** — the specific practice or set of practices

...ask one focused clarifying question before proceeding. If the input is sufficiently concrete, proceed directly to analysis.

## Step 3 — Produce the Structured Analytical Report

The report follows this structure:

---

### Report Structure

```
# Bourdieueva analiza: [Naziv pojave/prakse]

## 1. Pozicija u društvenom prostoru
Identifikacija relevantnih agenasa i njihova pozicija na dvema osama:
- Volumen kapitala (ukupan)
- Kompozicija kapitala (ekonomski vs. kulturni)

## 2. Kapitalni profil
Analiza po vrstama kapitala:
- Ekonomski kapital: ...
- Kulturni kapital (utelovljeni / objektivizovani / institucionalizovani): ...
- Socijalni kapital: ...
- Simbolički kapital (šta se naturalizuje kao zasluga): ...

## 3. Habitus
Rekonstrukcija habitusa:
- Koji uslovi egzistencije produkuju ove dispozicije?
- Koje sheme percepcije i vrednovanja su aktivne?
- Gde je habitus stečen (porodica / škola / polje)?

## 4. Polje
- U kom polju se odvija praksa?
- Kakva je specifična logika tog polja?
- Koje pozicije zauzimaju relevantni agensi unutar polja?

## 5. Ukus i logika distinkcije
- Kojoj od tri zone ukusa pripada ova praksa? (Legitimni / srednji / popularni)
- Ukus nužnosti ili ukus slobode?
- Kako praksa funkcioniše kao znak distinkcije (i prema gore i prema dole)?

## 6. Pogrešno prepoznavanje i simboličko nasilje
- Šta se naturalizuje kao prirodno, urođeno ili zasluženo?
- Ko prihvata tu naturalizaciju i zašto?
- Koje arbitrarne distinkcije prolaze kao univerzalne?

## 7. Doksa i iluzija
- Koje pretpostavke svi učesnici dele a ne propituju?
- Kakva je iluzija (ulaganje u igru) karakteristična za ovu praksu?

## 8. Reprodukcija
- Kako ova praksa doprinosi reprodukciji ili disrupciji društvene strukture?
- Koji mehanizam konverzije kapitala je aktivan?

## Zaključak
Sintetički uvid: šta Bourdieuev okvir otkriva o ovoj pojavi što bi ostalo nevidljivo bez njega?
```

## Step 4 — Standards for a Good Report

**Grounding:** Every major claim should be traceable to at least one Bourdieuev koncept iz referentnog fajla. Avoid vague sociological language — use the specific vocabulary (*habitus*, *illusio*, *misrecognition*, etc.).

**Relational thinking:** Bourdieu insists that positions only have meaning relationally. Always specify: *in relation to whom?* Distinctions only exist as differences.

**Avoid voluntarism:** Do not describe practices as if agents consciously chose them for strategic distinction. The logic of distinction is largely pre-reflective; practices are the product of habitus, not deliberate calculation (except in exceptional cases of conscious strategy, which themselves reflect a particular habitus).

**Avoid reductionism:** Cultural capital ≠ education level. Economic capital ≠ income. Use the full complexity of the three states of cultural capital and the capital composition axis.

**Language:** Write the report in the language the user used for the input. If the input was in Serbian, the report is in Serbian. Technical Bourdieu terms may be given in French/English in parentheses on first use.

**Length:** A full report is typically 600–1000 words across the 8 sections plus conclusion. For quick preliminary analyses, a condensed version (4 core sections + conclusion) is acceptable if the user requests it.
